Willkommen zum HTML Grundlagen Tutorial!

Dieses Tutorial hilft dir dabei, eine einfache Webseite mit grundlegenden HTML-Befehlen zu erstellen. Das ist eine gute Vorbereitung, um später mit PHP dynamische Webseiten zu bauen, da PHP oft HTML-Code generiert.

Die Aufgabe

Deine Aufgabe ist es, innerhalb von 70 Minuten eine eigene Webseite zu erstellen, die folgende Elemente enthält. Benutze dazu ausschließlich HTML-Code, den du selbst schreibst:

  • Unterschiedliche Schriftgrößen
  • Unterstrichener Text
  • Unterschiedliche Schriftfarben
  • Einrücken von Text
  • Einfügen einer Grafik
  • Verweis auf die Homepageseite der Schule
  • Zeilenumbrüche

Bereite dich darauf vor, dein Ergebnis der Klasse zu präsentieren.

Wichtig: Diese Seite ist rein statisch. Sie verwendet kein JavaScript oder serverseitige Sprachen wie PHP. Es geht nur um die Struktur und Darstellung mit HTML. Schreibe den Code selbst in einem Texteditor (z.B. Notepad++, VS Code, Sublime Text, o.ä.). Der <style>-Block oben dient nur der besseren Lesbarkeit dieses Tutorials!

HTML Grundlagen (Ganz kurz)

HTML steht für HyperText Markup Language. Es ist keine Programmiersprache, sondern eine Auszeichnungssprache. Sie beschreibt die Struktur einer Webseite mit Hilfe von sogenannten Tags.

Ein HTML-Tag besteht meistens aus einem öffnenden Tag (z.B. <p>) und einem schließenden Tag (z.B. </p>). Der Inhalt steht dazwischen.

Eine grundlegende HTML-Datei sieht so aus:

<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>Titel der Seite</title>
</head>
<body>
    <!-- Hier kommt der sichtbare Inhalt der Seite hin -->

</body>
</html>

Der <head> enthält Meta-Informationen (nicht direkt sichtbar), der <body> enthält den sichtbaren Inhalt.

Lösung der Aufgabe: Schritt für Schritt

1. Grundstruktur erstellen

Beginne mit der oben gezeigten Grundstruktur. Speichere die Datei mit der Endung .html (z.B. meine_seite.html).

2. Unterschiedliche Schriftgrößen

HTML bietet Überschriften-Tags von <h1> (größte) bis <h6> (kleinste).

<h1>Das ist die größte Überschrift</h1>
<h2>Das ist eine etwas kleinere Überschrift</h2>
<p>Das ist ein normaler Textabsatz.</p>
<h3>Noch kleiner...</h3>

Du kannst auch die Schriftgröße direkt mit dem style-Attribut ändern (fortgeschrittener, aber nützlich): 

<p style="font-size: 20px;">Dieser Text ist 20 Pixel groß.</p>
<p style="font-size: 10px;">Dieser Text ist nur 10 Pixel groß.</p>

Dieser Text ist 20 Pixel groß.

Dieser Text ist nur 10 Pixel groß.

3. Unterstrichener Text

Verwende den <u> Tag, um Text zu unterstreichen.

<p>Dieser Text ist <u>unterstrichen</u>.</p>

Dieser Text ist unterstrichen.

(Hinweis: Semantisch korrekter ist oft CSS mit style="text-decoration: underline;", aber für diese Aufgabe ist <u> einfacher und erlaubt).

4. Unterschiedliche Schriftfarben

Um die Farbe eines Textes zu ändern, verwendest du das style-Attribut direkt im HTML-Tag (z.B. in einem <p>-Tag). Innerhalb des style-Attributs nutzt du die CSS-Eigenschaft color gefolgt von einem Doppelpunkt und dem Farbwert. Farbwerte können Farbnamen (wie red, blue, green) oder Hexadezimal-Codes (wie #FF0000 für Rot, #0000FF für Blau) sein.

Hier ist der Code dazu:

<p style="color: red;">Dieser Text ist rot.</p>
<p style="color: blue;">Dieser Text ist blau.</p>
<p style="color: #008000;">Dieser Text ist grün (als Hex-Code).</p>
<p>Man kann auch nur <span style="color: orange;">einzelne Wörter</span> färben.</p>

Und so sieht es dann aus:

Dieser Text ist rot.

Dieser Text ist blau.

Dieser Text ist grün (als Hex-Code).

Man kann auch nur einzelne Wörter färben.

5. Einrücken von Text

Der <blockquote> Tag eignet sich gut, um ganze Textblöcke einzurücken (z.B. für Zitate).

<p>Normaler Text davor.</p>
<blockquote>
    Dieser Textblock wird vom Browser eingerückt dargestellt. Er eignet sich gut für längere Zitate oder hervorgehobene Abschnitte.
</blockquote>
<p>Normaler Text danach.</p>

Normaler Text davor.

Dieser Textblock wird vom Browser eingerückt dargestellt. Er eignet sich gut für längere Zitate oder hervorgehobene Abschnitte.

Normaler Text danach.

Für kleine Einrückungen am Anfang einer Zeile wird manchmal &nbsp; (geschütztes Leerzeichen) verwendet, aber das ist für größere Einrückungen unpraktisch und nicht empfohlen. CSS (padding-left oder text-indent) wäre hier die bessere Methode, aber für diese Aufgabe ist <blockquote> ausreichend. 

<!-- Beispiel mit &nbsp; (eher vermeiden) -->
<p>&nbsp;&nbsp;&nbsp;Dieser Text ist leicht eingerückt.</p>

   Dieser Text ist leicht eingerückt.

6. Einfügen einer Grafik

Verwende den <img> Tag. Dieser Tag hat keinen schließenden Tag. Wichtige Attribute sind src (Source = Quelle des Bildes) und alt (Alternativtext, falls das Bild nicht lädt oder für Screenreader). 

<!-- Stelle sicher, dass das Bild im selben Ordner liegt -->
<img src="cat.jpg" alt="Eine süße Katze" width="540" height="360">
Stelle sicher, dass sich die Bilddatei (z.B. cat.jpg) im selben Ordner wie deine HTML-Datei befindet, oder gib den korrekten Pfad an (z.B. src="bilder/logo.png"). Der alt-Text ist wichtig für Barrierefreiheit! Wenn du kein eigenes Bild hast, kannst du einen Platzhalter verwenden: <img src="https://placehold.co/540x360/EEE/31343C?text=Katzenbild" alt="Platzhalter Katzenbild">

8. Zeilenumbrüche

Normalerweise ignoriert HTML Zeilenumbrüche im Code. Wenn du einen Zeilenumbruch innerhalb eines Absatzes erzwingen willst, ohne einen neuen Absatz zu beginnen, nutze den <br> Tag (Break). Dieser hat keinen schließenden Tag. 

<p>Dies ist die erste Zeile.<br>
Dies ist die zweite Zeile, im selben Absatz.</p>

Dies ist die erste Zeile.
Dies ist die zweite Zeile, im selben Absatz.

Tipps für die Aufgabe

  • Speichern: Speichere deine Datei immer mit der Endung .html.
  • Testen: Öffne die HTML-Datei im Browser. Nach jeder Änderung: Speichern & Neu laden (F5/Cmd+R).
  • Bildpfad: Achte darauf, dass der Pfad im src-Attribut des <img>-Tags korrekt ist.
  • Schul-URL: Den richtigen Link zur Homepage einfügen
  • Selber tippen: Versuche, die Tags selbst zu schreiben, um sie besser zu lernen.
